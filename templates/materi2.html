Tipe Data & Tipe Data Abstrak

Dalam pemrograman, struktur data merupakan cara untuk mengorganisir dan menyimpan data agar dapat diakses dan dimodifikasi secara efisien. Beberapa struktur data telah didefinisikan secara langsung oleh bahasa pemrograman, seperti integer, string, dan array. Namun, ada juga struktur data yang tidak tersedia secara langsung dan harus didefinisikan oleh programmer sendiri. Struktur data yang didefinisikan oleh programmer ini disebut sebagai Abstract Data Type (ADT). ADT adalah konsep yang mendefinisikan cara kerja dan perilaku suatu struktur data tanpa menentukan implementasi spesifiknya.


Pengertian Tipe Data

Tipe data adalah kategori yang digunakan dalam pemrograman untuk menentukan jenis nilai yang dapat disimpan dan operasi-operasi yang dapat dilakukan pada nilai tersebut. Setiap bahasa pemrograman memiliki tipe data dasar yang dapat digunakan untuk menyimpan berbagai jenis informasi, seperti angka, teks, dan objek kompleks. Umumnya, tipe data dapat dikategorikan menjadi tiga, yaitu tipe data dasar (primitif), tipe data komposit, dan tipe data abstrak.

Jadi pada struktur data, tipe data menjadi dasar fondasi untuk membentuk struktur data. Banyak struktur data yang dikembangkan dari tipe data primitif dan komposit. Tipe data abstrak menjadi penghubung antara tipe data dan struktur data dengan menentukan tindakan atau operasi yang data dapat lakukan tetapi membiarkan penerapannya fleksibel.


Tipe Data Primitif

Tipe data primitif adalah tipe data dasar yang disediakan secara langsung oleh bahasa pemrograman dan tidak terdiri dari komponen yang lebih kecil. Tipe data ini digunakan untuk menyimpan nilai sederhana dan memiliki operasi bawaan yang dapat dilakukan tanpa perlu definisi tambahan (tidak perlu dibuat dalam kode sebelum digunakan). Tipe data primitif yang biasa ditemukan pada seluruh bahasa pemrograman diantaranya mencakup:

  • Bilangan Bulat (Integer) - Digunakan untuk menyimpan angka tanpa desimal, seperti int dalam Python, int dalam Java, atau short dan long dalam C++.
    Contoh : int x = 10
  • Bilangan Pecahan (Floating-Point) - Digunakan untuk menyimpan angka dengan desimal, seperti float dan double.
    Contoh : float y = 3.14
  • Karakter (Character) - Menyimpan satu karakter tunggal, seperti char dalam C dan Java.
    Contoh : char huruf = 'Z'
  • Boolean - Menyimpan nilai true atau false yang digunakan dalam operasi logika.
    Contoh : boolean isLogin = true
  • String - Menyimpan teks atau kumpulan karakter.
    Contoh : String matkul = "Praktikum SDBD"


Tipe data string dalam bisa dianggap sebagai primitif atau non-primitif tergantung pada bagaimana bahasa tersebut mengimplementasikannya.

Di beberapa bahasa seperti Python, JavaScript, dan Java, string sering dianggap primitif karena tipe data string langsung oleh bahasa pemrograman.

Namun, di bahasa seperti C dan C++, string dianggap non-primitif, karena string tidak disediakan langsung. String pada C dan C++ biasanya direpresentasikan dengan array berisi char.

Berbeda dengan bahasa lain, bahasa python menyediakan fitur pendeteksian tipe data otomatis sehingga kita tidak perlu menuliskan tipe data pada variabel. Fitur ini disebut Dynamic Typiing Python akan menentukan tipe data berdasarkan isi dari variabel. Untuk keterangan lebih lanjut, teman-teman dapat membuka link berikut : Tipe Data Python W3Schools
Kelebihan dari sitem dynamic typing adalah penulisan variabel menjadi lebih sederhana sehingga pemrograman lebih mudah. Namun, hal tersebut memberikan kekurangan yaitu tipe data yang tidak dapat kita definisikan sendiri.


Apa yang mendefinisikan tipe data primitif ?

Jadi sederhananya tipe data primitif adalah tipe data bawaan yang sudah disediakan oleh bahasa pemrograman.

Jika seandainya ada tipe data khusus yang hanya ada di bahasa pemrograman tertentu namun merupakan bawaan dari suatu bahasa pemrograman, maka tipe data tersebut dapat dikategorikan sebagai tipe data primitif.

Contoh sederhananya adalah Vector2(x,y), yaitu tipe data yang berisi koordinat x dan y yang biasa digunakan pada pemrograman game atau visual grafis.


Tipe Data Komposit

Tipe data komposit adalah tipe data yang mampu menyimpan lebih dari satu nilai dalam satu variabel dengan berbagai cara pengorganisasian data. Tipe ini digunakan untuk mengelola kumpulan data yang memiliki hubungan atau relasi.


Esensi Data Komposit

Untuk memahami data komposit lebih lanjut, berikut adalah poin-poin utama dari data komposit :


Dapat menyimpan lebih dari satu nilai

Berbeda dengan tipe primitif yang hanya menyimpan satu nilai, tipe data komposit dapat menyimpan beberapa nilai dalam satu variabel.
Contoh : 
angka = [10, 20, 30, 40, 50] #Array tipe List pada Python
print(angka)  # Output: [10, 20, 30, 40, 50]

koordinat = (12.5, 45.7)  # Array tipe Tuple pada Python
print(koordinat)  # Output: (12.5, 45.7)

Nilai dapat memiliki suatu hubungan atau relasi

Data dalam tipe komposit sering memiliki keterkaitan antar elemen. Oleh karena itu, data ini dapat disusun dalam berbagai bentuk, seperti:
  • Linier, yaitu struktur data yang menyusun elemen secara berurutan seperti Array dan Linked List. Contoh : 
    antrian = ["Oberon", "Lysander", "Hermia"]  # Elemen disini disusun secara linier.
print(antrian[0])  # Output: Oberon
  • Hierarkis, Struktur data yang memiliki hubungan parent-child atau node (titik) yang saling terhubung. Contoh : 
    # Membuat struktur pohon dengan dictionary
tree = {
    "A": ["B", "C"],    # A memiliki dua anak: B dan C
    "B": [],          # B tidak memiliki anak
    "C": []          # C tidak memiliki anak
}

# Cetak root (akar)
print(tree.keys())  # Output: dict_keys(['A', 'B', 'C']).
# Ini maksudnya pohon tersebut memiliki tiga node yaitu A, B, dan C.

# Cek anak dari "A"
print(tree["A"])  # Output: ['B', 'C']
# Ini maksudnya ketika kita mengakses 'A', pilihan 'B' dan 'C' muncul.
  • Tabular, Data disusun dalam format baris dan kolom seperti dalam tabel database atau DataFrame di Python. 
    import pandas as pd
# Membuat data frame dengan 3 baris dan 3 kolom
df = pd.DataFrame({
    'Nama': ['A', 'B', 'C'],
    'Umur': [20, 21, 22],
    'Alamat': ['Jl. A', 'Jl. B', 'Jl.C']
    })

    # Menampilkan data frame
    print(df)
    Hasilnya :
    Index Nama Umur Alamat
    0 A 20 Jl. A
    1 B 21 Jl. B
    2 C 22 Jl. C
  • Asosiatif, Data disusun dalam pasangan kunci-nilai (key-value pairs) untuk pencarian cepat. Contoh : 
    mahasiswa = {
    "NIM001": {"Nama": "Azmi", "Jurusan": "TI"},
    "NIM002": {"Nama": "Thariq", "Jurusan": "RPL"}
}

print(mahasiswa["NIM001"]["Nama"])  # Output: Azmi

Apakah Nama = [Vortigern] adalah variabel dengan tipe data komposit?

Nama = [Vortigern] adalah tipe data komposit walaupun variabel 'Nama' hanya memiliki satu nilai.

Hal tersebut disebabkan karena variabel 'Nama' sudah bertipe data Array walaupun isinya hanya satu. Jadi, variabel tipe data komposit tetap sah walaupun nilainya hanya satu jika jenis variabel sudah ditentukan sebelumnya. (disini variabel array ditentukan dengan adanya kurung siku [])

Vortigern disini adalah suatu nilai yang berada di variabel 'Nama'

Tipe Data Abstrak

Tipe Data Abstrak (ADT) adalah konsep model untuk struktur data yang mendefinisikan operasi-operasi yang dapat dilakukan pada data tanpa menentukan bagaimana operasi tersebut diimplementasikan. ADT memberikan kerangka struktur untuk apa yang data bisa lakukan dan apa yang data tidak bisa lakukan.

Karena ADT hanya berfokus pada logika operasi dan tidak bergantung pada cara kerja bahasa pemrograman, konsep ini dapat diterapkan di berbagai bahasa pemrograman. Implementasi ADT dibuat secara langsung dalam kode oleh pengembang, karena bahasa pemrograman umumnya tidak menyediakan ADT secara langsung dan hanya menyediakan alat atau tipe data dasar untuk membangunnya.


Contoh Penerapan ADT

Berikut adalah salah satu contoh bagaimana ADT diguunakan pada suatu program : 
class Stack:
def __init__(self):
    self.stack = []  # Menggunakan list untuk menyimpan elemen stack

def push(self, data):
    # Menambahkan elemen ke dalam stack
    self.stack.append(data)

def pop(self):
    # Menghapus dan mengembalikan elemen teratas dari stack
    if not self.is_empty():
        return self.stack.pop()
    return "Stack kosong"

def peek(self):
    # Mengembalikan elemen teratas tanpa menghapusnya
    if not self.is_empty():
        return self.stack[-1]
    return "Stack kosong"

def is_empty(self):
    # Mengecek apakah stack kosong
    return len(self.stack) == 0

def size(self):
    # Mengembalikan jumlah elemen dalam stack
    return len(self.stack)

def display(self):
    # Menampilkan elemen stack
    print(self.stack[::-1])  # Menampilkan dalam urutan terbalik (elemen teratas dulu)

# Contoh penggunaan
s = Stack()
s.push(10)
s.push(20)
s.push(30)
s.display()  # Output: [30, 20, 10]

print(s.pop())  # Output: 30
print(s.peek())  # Output: 20
print(s.is_empty())  # Output: False
